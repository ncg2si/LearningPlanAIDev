# LearningPlanAIDev
#LLM/RAG learning plan tracker with daily plans and tutorials
## Summary
- Adds a static web app for the 12-week full-time LLM/RAG learning path toward AI developer roles.
- Includes 72 day-by-day schedules (Mon–Sat), curated tutorials with links and learning steps, and local progress tracking.
- Provides week/day navigation, a today shortcut based on start date, interview prep, and application checklist views.
